# virustotal-hash-scanner
python tool to check file hashes with VirusTotal API. You can scan single or multiple hashes and see which hashes are malicious, suspicious, harmless or undetected.

#Features
-scan one or multiple file hashes.
-Accept hashes from user input or a file ('hash.txt")
-Show total votes: malicious, suspicious, harmless, undetcted.
-Automatically detect if hash is malicious, suspicious or safe.

#Requirements
-python 3
-requests library('pip install requests')

#How to Use 
1 Run the python script: 
python virustotal.py
